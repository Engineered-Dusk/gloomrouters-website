# Gloom Routers Website Guardrails

This is the public static website/deploy repo for Gloom Routers.

Preserve these public routes: `/`, `/cards/`, `/story/`, and `/play/`.

`/play/` contains generated static deploy output. Avoid hand-editing generated
output except for emergency static deploy fixes.

Do not commit local reports, temp files, credentials, source maps, private paths,
or personal data.

Do not copy local reports, RUN_REPORT.md files, audit reports, temp clone notes, or private runbooks into this public repo.

Keep changes narrow and public-safe.

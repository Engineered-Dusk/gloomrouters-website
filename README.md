# Gloom Routers Website

This repository contains the public static website for **Gloom Routers**.

## Public Site

- `index.html`: landing page.
- `styles.css`: shared website styling.
- `cards/`: static card browser.
- `story/`: public story pages.
- `play/`: generated static deploy output for the playable browser prototype.
- `assets/`: public website and game assets.
- `CNAME`: custom domain for GitHub Pages.

Game-source changes should be made outside this public repo and deployed here as
rebuilt static output.

## Local Preview

Open `index.html` directly in a browser or serve the repository with any simple
static file server.
